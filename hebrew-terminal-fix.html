פתרון בעיות

הכפתור "Open in Terminal" לא פותח טרמינל?

במקרים נדירים אחרי התקנת גרסה ישנה, הכפתור מפסיק לעבוד. הפתרון — Reset לאפליקציה:

1

פתחו Settings (מקש Windows + I)

פתחו Start → Settings
2

לכו ל-Apps

בחרו Apps
3

לחצו על Installed Apps

לחצו Installed Apps
4

חפשו "Terminal" (או "מסוף" בעברית) → לחצו ⋯ → Advanced options

חפשו Terminal → ⋯ → Advanced options
5

לחצו על Reset — וסיימתם!

לחצו Reset

זהו! הכפתור "Open in Terminal" אמור לעבוד עכשיו. לא צריך הפעלה מחדש.

העברית חזרה להיות הפוכה?

Windows מעדכן אוטומטית את Terminal לגרסה חדשה. זה צפוי לקרות — אל דאגה:

  1. התקינו שוב את Terminal 1.18 מהקובץ (כפתור ההורדה למעלה)
  2. אם הכפתור לא פותח טרמינל — עשו Reset (לפי השלבים למעלה)

טיפ: שמרו את קובץ ההתקנה במקום נגיש לשימוש עתידי.
